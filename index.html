Connect to GitHub Gist

Your filament data is stored privately in a GitHub Gist — only you can see it.

GitHub → Settings → Developer settings → Personal access tokens (classic) → tick gist
The long code at the end of your Gist URL: gist.github.com/username/→ this part ←
Step-by-step: how to create a Gist and token
Create a secret Gist
  1. Go to gist.github.com (logged into GitHub)
  2. In "Gist description" type: Filament Tracker data
  3. In "Filename…" type inventory.csv — leave content blank
  4. Click + Add file → type purchases.csv
  5. Click + Add file → type stocktakes.csv
  6. Click the dropdown arrow on the green button → Create secret gist
  7. Copy the long code from the URL after your username — that's your Gist ID
Create a token
  1. Profile photo → Settings → scroll to bottom → Developer settings
  2. Personal access tokens → Tokens (classic)
  3. Generate new token (classic) → name it anything
  4. Tick the gist checkbox only
  5. Click Generate token → copy it (starts with ghp_)
  6. Save it somewhere safe — you can only see it once
